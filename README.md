课程介绍：

测试界的清华!!!

1983年，IEEE提出的软件工程标准术语，软件测试定义为：“使用人工和自动化工具来测试或运行某个系统的过程，其目的是在于检验它是否满足规定的需求或者弄清预期结果与实际结果之间的差别”

通俗说：软件测试就是在软件投入运行之前，对软件需求分析，设计规格说明书和编码实现的最终审查，它是软件质量保证的关键步骤。
